02) pwd
 show present working directory

03) ls
 list the files and directories

04) mkdir <name>
 Creates a new directory with the specified name.

05) ls -la
 Lists all files and directories (including hidden ones) in long format.

06)  git log
 Displays the commit history for the current branch


07) cd <folder_name>
 Changes the current directory to the specified folder.

08) cd ..
  back one level in directory hierarchy.

09) git add .
 Stages all changes in current directory.

10) rm <file/folder_name>
 Removes the specified file or folder.

11) rm -rf <folder_name>
 Permanently removes a folder and its contents.!!!!!!!be careful!!!!!!!

12) git commit -m "message"
 creates a new commit with a brief commit message
